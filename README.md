# CellLook_Shader
ワンクリックでセルルックのシェーダーを生成するアドオンです

## インストール方法
1. Edit（編集）> 環境設定（Preferences）> Add-ons
2. Add-onsの中に、zipファイルごとドラック＆ドロップ

## 使い方
- 任意のオブジェクトを選択
- シェーダーエディターの右側にある「toon」のタブをクリック
  - 表示されてなければ「N」を押して表示
- 「Create Cell Look Material」をクリック
- ノードが自動で生成されます。いじるところは下記。
  - Ramp Pos：色の境界の閾値を調整
  - カラー：AとBの2色を設定
  - Mix：プルダウンからブレンドモードを設定
- Read Meには、おすすめの設定を記載しているので参考に
